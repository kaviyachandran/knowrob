entity(?Entity, +Descr) is nondet
Query for entity matching an entity description or build description for entity.
Arguments:
Entity- IRI of matching entity
Descr- An entity description, OWL individual or class IRI
entity_has(+Descr, +Key, Value)
Read entity description value
entity_assert(-Entity, +Descr)
Assert entity description in RDF triple store as new individual.
Arguments:
Entity- IRI of matching entity
Descr- An entity description
entity_retract(+Entity)
Assert entity description in RDF triple store as new individual.
Arguments:
Entity- IRI of matching entity
with_owl_description(+Description, ?Individual, +Goal)
Ensures entity description is asserted and binds the name to Individual before goal is called. Temporary assertions are retracted in a cleanup goal.
Arguments:
Description- Entity description or individual
Individual- Entity individual Goal The goal with OWL entity asserted
entity_write(+Descr) is nondet
Write entity to current output stream.
Arguments:
Descr- An entity description
