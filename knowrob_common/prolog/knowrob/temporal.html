assert_temporal_part_end(+S, ?P, ?O) is nondet
assert_temporal_part_end(+S, ?P, ?O, +End) is nondet
assert_temporal_part_end(+S, ?P, ?O, +End, +Graph) is nondet
Assert that a given relation `P(S,O)` stopped holding starting from specified time instant End (or current time if End is not specified). The assertion specifies the endTime of the temporal extend of the temporal part that describes this relation. Additionally any non-temporal representation of this relation will be retracted (so that standard reasoners won't yield this relation anymore).
Arguments:
S- The subject of the relation
P- The relation predicate
O- The object of the relation or a data-typed value
End- The time instant at which the relation stopped holding
Graph- The RDF graph name
