rdf_instance_from_class(+Class, -Inst) is nondet
Utility predicate to generate unique instance identifiers
Arguments:
Class- Class describing the type of the instance
Inst- Identifier of the generated instance of Class
rdf_instance_from_class(+Class, +SourceRef, -Inst) is nondet
Utility predicate to generate unique instance identifiers using the source reference SourceRef for rdf_assert
Arguments:
Class- Class describing the type of the instance
SourceRef- Atom as source reference for rdf_assert
Inst- Identifier of the generated instance of Class
rdf_unique_id(+Class, -UniqID) is det
UniqID is a IRI that uses Class as prefix and is not yet used in the RDF triple store.
Arguments:
Class- Class IRI
UniqID- Unused IRI with prefix Class
rdf_phas(+Property, ?P, ?O) is nondet
True if Property or one of its super properties has a property assertion for P with value O.
Arguments:
Property- A RDF property
P- A RDF property
O- A RDF resource
rdf_has_prolog(?Subject, ?Predicate, ?Value)
Calls rdf_has and converts data values to Prolog representation (e.g., xsd:float to Prolog number).
Arguments:
Subject- RDF resource iri
Predicate- RDF predicate iri
Value- RDF iri or Prolog encoded data value
rdf_assert_prolog(?Subject, ?Predicate, ?Value)
rdf_assert_prolog(?Subject, ?Predicate, ?Value, +Graph)
Asserts a triple to the RDF datastore, and accepts some Prolog encoded data values.
Arguments:
Subject- RDF resource iri
Predicate- RDF predicate iri
Value- RDF iri or Prolog encoded data value
