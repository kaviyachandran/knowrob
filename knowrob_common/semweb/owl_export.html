owl_export.pl -- Methods for exporting triples into OWL files

This module contains methods for exporting triples into OWL files, for instance object definitions, environment maps, or task specifications.

export_object(+Obj, -File)
Export the perception of an object to an OWL file. Not-existing directories will be created.
Arguments:
Obj- Object instance to be exported (including the perception instances)
File- Filename for the exported OWL file
export_object_class(+Obj, -File)
Export the definition of an object class to an OWL file. Not-existing directories will be created.
Arguments:
Obj- Object class to be exported
File- Filename for the exported OWL file
export_map(+Map, -File)
Export the map as the set of all perceptions of objects to an OWL file. Not-existing directories will be created.
Arguments:
Map- Map instance to be exported (including the perception instances for all objects)
File- Filename for the exported OWL file
export_action(+Action, -File)
Export an action specification (TBOX) to an OWL file. Not-existing directories will be created.
Arguments:
Action- Action specification to be exported
File- Filename for the exported OWL file
