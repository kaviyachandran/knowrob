marker_primitive(+Type, +MarkerName, +MarkerTerm, -MarkerObject, +Parent) is det
Creates MarkerObject instance with primitive geometry shape (e.g., cube, sphere, ...) and some default properties.
Arguments:
Type- The shape primitive type (defined by marker_prop_type/2)
MarkerName- A atom that identifies the new marker (e.g., 'PR2')
MarkerTerm- A term that identifies the new marker (e.g., trajectory('/base_link'))
MarkerObject- The MarkerObject object created
Parent- The parent Java object (i.e., MarkerPublisher or MarkerObject instance)
marker_remove(+MarkerTerm) is det
Removes MarkerObject instances that match given term. marker_remove(all) removes all known markers. marker_remove(trajectories) removes all known trajectory markers. marker_remove(MarkerTerm) removes all markers that can be unified with the compound term MarkerTerm.
Arguments:
MarkerTerm- Marker identification term.

MarkerTerm- A term that identifies the marker (e.g., trajectory('/base_link'))
MarkerChildTerm- A term that identifies the marker child

MarkerObject- The MarkerObject instance

Find all query representations that can be asked about this marker. These are queries described in an OWL ontology that can be asked about particular classes or individuals. Queries is a list of [atom Category,Title,Query].
