map_root_objects(Map, Objs) is nondet
Read the 'root objects' in the map, i.e. those that are asserted as direct children of the map instance (e.g. cupboards, drawers)
Arguments:
Map- Instance of a knowrob:SemanticEnvironmentMap
Objs- List of instances of the root objects in the map
map_root_object(Map, Obj) is nondet
Read the 'root objects' in the map, i.e. those that are asserted as direct children of the map instance (e.g. cupboards, drawers)
Arguments:
Map- Instance of a knowrob:SemanticEnvironmentMap
Objs- Instance of a root object in the map
map_child_objects(+Parent, -Children) is nondet
Read all object instances asserted as properPhysicalParts of Parent
Arguments:
Parent- Object instance
Children- List of object instances asserted as physical part of Parent
map_child_object(?Parent, ?Child) is nondet
Read all object instances asserted as properPhysicalParts of Parent
Arguments:
Parent- Object instance
Child- Object instance
map_object_info(-Info) is nondet
Read information about all objects in the map. Returning a structured list for each object consisting of the instance ID, the class, the pose, and a list of the object dimensions [D, W, H].
Arguments:
Inst- Object instance
map_object_type(?Obj, ?Type) is nondet
Read the type of an object instance
Arguments:
Obj- Object instance
Type- Class of Obj
map_object_label(?Obj, ?Label) is nondet
Reads the rdfs:label of an instance
Arguments:
Obj- Object instance
Label- Value of the rdfs:label annotation
map_object_dimensions(Obj, D, W, H) is nondet
Read the dimensions (depth, width, height) of an object.
Arguments:
Obj- Object instance
D- Object depth (x dimension)
W- Object Width (y dimension)
H- Object height (z dimension)
map_connection_type(Parent, Child, ConnectionType) is nondet
Determine the type of connection between Parent and Child, e.g. the type of joint.
Arguments:
Parent- Object instance
Child- Object instance
ConnectionType- Type of the connecting element, e.g. 'HingedJoint', 'PrismaticJoint', or 'Fixed'
