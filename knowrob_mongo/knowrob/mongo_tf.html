mng_lookup_transform(+TargetFrame, +SourceFrame, -Pose, +Instant) is nondet
Determine the transform from Source to Target at time Instant based on the logged tf data.
Arguments:
TargetFrame- Target frame ID
SourceFrame- Source frame ID
Pose- The pose in the form of pose([float x,y,z],[float qx,qy,qz,qw])
Instant- The time instant
mng_transform_pose(+SourceFrame, +TargetFrame, +PoseIn, -PoseOut, -Instant) is nondet
Transform a pose into TargetFrame at time Instant.
Arguments:
SourceFrame- Tf frame the source matrix is described in
TargetFrame- Tf frame the source matrix is to be transformed into
PoseIn- Term of form pose([float x,y,z],[float qx,qy,qz,qw])
PoseOut- Term of form pose([float x,y,z],[float qx,qy,qz,qw])
Instant- Time point at which the transformation is to be determined
mng_comp_pose(+Obj, -Pose, +Interval)
Pose is an OWL individual of knowrob:'Pose' that represents logged tf data.
