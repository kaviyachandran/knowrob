mng_designator(+Designator, -DesigJava) is semidet
Read object that corresponds to Designator into a JAVA object DesigJava. Designator is either a java mongo DB object, or an OWL individual of type knowrob:'Designator'.
Arguments:
Designator- The designator DB object or OWL individual
DesigJava- Java designator object
mng_designator_type(+Designator, ?Type) is semidet
True if Type is the type of Designator. This is usually "object", "action", or "location".
Arguments:
Designator- Java designator object or OWL individual of knowrob:Designator
Type- Type of the designator
mng_designator_timestamp(+Designator, ?Instant) is semidet
True if Instant is the time at which Designator was created.
Arguments:
Designator- Java designator object or OWL individual of knowrob:Designator
Instant- Floating point value representing the time
mng_designator_interval(+Designator, ?Interval) is nondet
True if Interval is the time interval during which Designator was the most recent representation of an entity.
Arguments:
Designator- Java designator object or OWL individual of knowrob:Designator
Interval- [float begin, end] or [float begin]
mng_designator_location(+Designator, ?Pose) is semidet
True if Pose is the location of Designator.
Arguments:
Designator- Java designator object or OWL individual of knowrob:Designator
Pose- The location pose in form of pose([x,y,z],[float qx,qy,qz,qw])
mng_designator_object_type(+Designator, ?Type) is nondet
Read the type of a logged designator by its ID
Arguments:
Designator- Java designator object or OWL individual of knowrob:Designator
Type- The type of the object that corresponds to Designator
mng_designator_property(+Designator, ?Key, ?Value) is nondet
Read the properties of a logged designator by its ID
Arguments:
Designator- Java designator object or OWL individual of knowrob:Designator
Key- List of property keys for nested designators
Value- The property value
mng_designator_resolve(+Designator, ?Entity) is nondet
True if Entity is a valid representation for Designator.
Arguments:
Designator- Java designator object or OWL individual of knowrob:Designator
Entity- OWL indiviudal that represents the designator
