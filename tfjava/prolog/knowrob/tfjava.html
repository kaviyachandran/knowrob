tfjava.pl -- Prolog-predicates as wrappers around tfjava

Provides an interface to the tf system from within Prolog, allowing to transform coordinates and to lookup transformations.

tfjava_start_listener is det
Starts the tf listener, which runs in a separate thread.
tfjava_lookup_transform(+TargetFrame, +SourceFrame, -Pose, +Instant) is nondet
Lookup transform of SourceFrame relative to TargetFrame at time Instant.
Arguments:
TargetFrame- Tf frame the source matrix is to be transformed into
SourceFrame- Tf frame the source matrix is described in
Pose- Term of form pose([float x,y,z],[float qx,qy,qz,qw])
Instant- Time point at which the transformation is to be determined
tfjava_transform_point(+SourceFrame, +TargetFrame, +PointIn, -PointOut, -Instant) is nondet
Transform a point into TargetFrame at time Instant.
Arguments:
SourceFrame- Tf frame the source matrix is described in
TargetFrame- Tf frame the source matrix is to be transformed into
PointIn- List of form [float x,y,z]
PointOut- List of form [float x,y,z]
Instant- Time point at which the transformation is to be determined
tfjava_transform_pose(+SourceFrame, +TargetFrame, +PoseIn, -PoseOut, -Instant) is nondet
Transform a pose into TargetFrame at time Instant.
Arguments:
SourceFrame- Tf frame the source matrix is described in
TargetFrame- Tf frame the source matrix is to be transformed into
PoseIn- Term of form pose([float x,y,z],[float qx,qy,qz,qw])
PoseOut- Term of form pose([float x,y,z],[float qx,qy,qz,qw])
Instant- Time point at which the transformation is to be determined
