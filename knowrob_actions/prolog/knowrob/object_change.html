transformed_into(?From:iri, ?To:iri)
Compute which objects have been transformed into which other ones by actions or processes. This predicate operates on the object modification graph created by the action projection rules
Arguments:
From- Input of some action
To- Output created by this action
transformed_into_transitive(?From:iri, ?To:iri)
Transitive version of the transformed_into predicate that tracks in- and outputs of actions over several steps and different properties
Arguments:
From- Input of some action
To- Output created by this action
comp_thermicallyConnectedTo(?Obj1:iri, ?Obj2:iri)
Compute if a heat path exists between two objects. This is the case if they are either on top of each other or if one contains the other one
Arguments:
Obj1- Object instance
Obj2- Object instance
