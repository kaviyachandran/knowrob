action_objectActedOn(?Action, ?Object) is nondet
Reads the objectActedOn for a TBOX action description
Arguments:
Action- Action class with a restriction on the objectActedOn
Object- Value set as objectActedOn for this action

action_inputs(+Action, -Input)
Required inputs for an action
Arguments:
Action- Action class
Input- Input linked via a preActors restriction
action_missing_inputs(+Action, -Missing)
Missing inputs of an action (required, but not available)
Arguments:
Action- Action class
Missing- Input linked via a preActors restriction, but not available
action_outputs(+Action, -Output)
Outputs of an action
Arguments:
Action- Action class
Output- Output linked via a postActors restriction
